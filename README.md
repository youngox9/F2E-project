## 使用技術

## 使用套件

- GSAP
- ScrollMagic
- styled-component

## 資料夾結構

```

│── pages/  -> [頁面]
│── public/  -> [圖片,字型都放這]
├── src
│   ├── components/  -> [組件]
│   ├── hook/   -> [重要hook邏輯都放這裡]
│   ├── store/  -> [redux store]
│   ├── styles/  -> [主要scss樣式都在這裡]
│
│── next.config.js  -> [Next主要設定檔]

```

## 運行方式

```
npm install
npm run dev  
```
