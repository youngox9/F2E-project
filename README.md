## 使用技術

- GSAP

## 資料夾結構

```
│── pages 頁面
│── public/ 圖片,字型都放這
│── app.min.js (效果相關)
├── src
│   ├── components 組件
│   ├── hook 重要hook邏輯都放這裡
│   ├── store redux store
│   ├── styles.scss 主要scss
```

## 運行方式

```
npm install
npm run dev
```
